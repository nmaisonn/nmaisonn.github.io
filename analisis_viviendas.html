Algoritmo Utilizado para Predicción del Precio de las Viviendas

Regresión Lineal

Descripción:

La regresión lineal es un algoritmo de aprendizaje automático utilizado para modelar la relación entre una variable dependiente (la variable que queremos predecir) y una o más variables independientes (atributos) mediante una ecuación lineal. El objetivo es encontrar la línea recta (en el caso de una sola característica) o un hiperplano (en el caso de múltiples características) que mejor se ajuste a los datos.

Entradas:

Variables Independientes: Son las entradas del modelo y representan las características o atributos de los datos. Pueden ser una o varias, dependiendo de la complejidad del problema.

Salidas:

Variable Dependiente (Variable Objetivo): Es la salida del modelo y representa la variable que se desea predecir.

Cuándo se Usa:

La regresión lineal se utiliza cuando se sospecha que existe una relación lineal o aproximadamente lineal entre las características y la variable objetivo. Es adecuada para problemas de regresión, donde se busca predecir un valor numérico continuo, como el precio de una casa, los ingresos de una persona o la temperatura.

Tipo de Algoritmo:

La regresión lineal es un algoritmo de aprendizaje supervisado, ya que se basa en datos etiquetados para aprender la relación entre las variables.

Utilidad:

La regresión lineal es ampliamente utilizada en estadísticas y aprendizaje automático para modelar y predecir relaciones lineales en datos. Es útil para comprender la relación entre las variables y hacer predicciones basadas en esa relación. Puede ser utilizado en aplicaciones prácticas como la predicción de precios de bienes raíces, análisis de tendencias económicas, pronóstico del tiempo y más.

Elección del algoritmo:

Previamente, en los primeros pasos del análisis del caso de estudio. Se realizó una investigación para ver distintos algoritmos que fueran comúnmente utilizados para casos de predicción de precios a partir de una gran cantidad de atributos. Por ello, llegué a la conclusión que la regresión lineal es una técnica ampliamente utilizada en problemas de predicción de precios, y sus beneficios son clave en este tipo de contextos. En primer lugar, la regresión lineal es adecuada cuando existe una relación lineal aparente o razonable entre las características de entrada y la variable objetivo, en este caso, el precio de las viviendas. Esto significa que, si hay una correlación o una tendencia lineal entre las características seleccionadas y el precio de las viviendas, la regresión lineal puede capturar y cuantificar esta relación de manera efectiva. La filtración y selección cuidadosamente las características relevantes ayudó a asegurar que la relación lineal fuera más evidente, lo que respalda aún más la elección de la regresión lineal. Además, la regresión lineal es computacionalmente eficiente y fácil de implementar, lo que facilita su aplicación y ajuste en grandes conjuntos de datos, como los relacionados con bienes raíces.
